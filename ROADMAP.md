# Roadmap

## Phase 1: Transparent Baselines

- Keep all agent workflows runnable without API keys.
- Save tool traces, retrieval traces, and evaluation results.
- Document failure cases for planning, grounding, and escalation.

## Phase 2: LLM Integration

- Replace rule-based planners with an API or local LLM planner.
- Add structured tool/function calling.
- Add memory and state persistence.
- Compare multiple prompts and model versions.

## Phase 3: Framework Upgrade

- Rebuild the research assistant as a LangGraph-style state machine.
- Rebuild the support workflow as a multi-agent collaboration system.
- Add automated regression tests for agent traces.

## Phase 4: Portfolio Polish

- Add screenshots and output figures.
- Add experiment reports and failure analysis.
- Add deployment notes for Streamlit or FastAPI demos.
